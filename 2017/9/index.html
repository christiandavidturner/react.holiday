Day 9: Set state dynamically

If you want to set state dynamically, use a function.

this.setState(function(previousState) {
  return { someProp: previousState.someProp + 1 }
})

We give setState an anonymous function to be called with state. Our function takes that state and returns a new object with new values derived from the old.
